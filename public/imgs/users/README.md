# Fotos de Perfil dos Usuários

Esta pasta contém as fotos de perfil dos usuários (alunos) utilizadas nos mockups.

## Estrutura de Nomes

As fotos devem seguir o padrão de nomenclatura:
- `user-{id}.jpg` ou `user-{id}.png`
- Ou simplesmente `{id}.jpg` / `{id}.png`

Onde `{id}` corresponde ao ID do usuário/aluno.

## Formato Recomendado

- **Formato**: JPG ou PNG
- **Tamanho**: 300x300px ou maior (quadrado)
- **Qualidade**: Boa qualidade
- **Estilo**: Profissional e adequado para perfil

## Como Adicionar Fotos

1. Adicione as fotos nesta pasta seguindo o padrão de nomenclatura acima
2. O código está configurado para usar essas imagens automaticamente
3. Certifique-se de que os arquivos tenham extensão `.jpg`, `.jpeg` ou `.png`

## Nota

As fotos podem ser usadas tanto para alunos quanto para outros tipos de usuários do sistema.
