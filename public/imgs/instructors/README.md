# Fotos dos Instrutores

Esta pasta contém as fotos dos instrutores utilizadas nos mockups.

## Estrutura de Nomes

As fotos devem seguir o padrão de nomenclatura:
- `instructor-{id}.jpg` ou `instructor-{id}.png`

Onde `{id}` corresponde ao ID do instrutor no arquivo `mockData.js`.

## Formato Recomendado

- **Formato**: JPG ou PNG
- **Tamanho**: 300x300px ou maior (quadrado)
- **Qualidade**: Boa qualidade, profissionais e sérias
- **Faixa etária**: 35-60 anos
- **Estilo**: Profissional, mas não muito formal

## Como Adicionar Fotos

1. Adicione as fotos nesta pasta seguindo o padrão de nomenclatura acima
2. O código já está configurado para usar essas imagens automaticamente
3. Certifique-se de que os arquivos tenham extensão `.jpg`, `.jpeg` ou `.png`
